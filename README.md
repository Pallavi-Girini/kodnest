# Job Notification Tracker - KodNest Premium

A job notification tracking application built with React and React Router, following the KodNest Premium Design System.

## Design System

- **Background**: Off-white (#F7F6F3)
- **Accent Color**: Deep red (#8B0000)
- **Typography**: Serif headings (Playfair Display), Sans-serif body (Inter)
- **Layout**: Calm, high whitespace design

## Routes

- `/` - Home (redirects to Dashboard view)
- `/dashboard` - Dashboard
- `/saved` - Saved notifications
- `/digest` - Digest view
- `/settings` - Settings
- `/proof` - Proof section

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
